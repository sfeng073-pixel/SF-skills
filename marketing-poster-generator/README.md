# marketing-poster-generator

> SOLO 营销海报生成技能 — 市场调研 → 文案设计 → 视觉生成 → 迭代优化

## 简介

`marketing-poster-generator` 是一个 SOLO AI 技能，为营销团队提供**标准化的海报生成流程**。当用户需要制作营销海报时，技能会自动执行市场调研、本地化文案设计、AI 图片生成和质量验证，确保输出的海报符合目标市场的文化和审美习惯。

## 功能特性

- **市场调研先行**：自动搜索同类海报趋势、本地化要点、竞品参考
- **本地化适配**：内置台湾繁体/大陆简体/英文语言规范和用词转换
- **标准化流程**：6 步流程确保每次输出质量一致
- **质量验证**：生成后自动检查品牌、文字、形象、水印等 6 项指标
- **迭代优化**：最多 5 次迭代，内置常见问题处理方案

## 流程

```
用户输入主题 → 市场调研 → 调研报告 → 沟通细节 → 文案设计 → 生成海报 → 迭代优化 → 交付
```

| 步骤 | 内容 | 工具 |
|------|------|------|
| 1. 市场调研 | 搜索海报趋势、本地化要点、竞品参考 | WebSearch + WebFetch |
| 2. 沟通细节 | 确认品牌、卡通形象、活动内容、视觉偏好 | AskUserQuestion |
| 3. 文案设计 | 设计 2-3 套本地化文案方案 | - |
| 4. 生成海报 | 按 Prompt 模板生成图片 | GenerateImage |
| 5. 迭代优化 | 验证 + 修正，最多 5 次 | Read + GenerateImage |
| 6. 交付 | 输出海报文件 + 评估报告 | - |

## 安装

将 `marketing-poster-generator` 文件夹放入 SOLO 技能目录：

```bash
# SOLO 技能目录
~/.solo/skills/marketing-poster-generator/
# 或
/mnt/appuserdata/skills/marketing-poster-generator/
```

## 使用方法

在 SOLO 中直接描述需求即可触发技能：

```
帮我生成一个 VIPTHINK 618 暑期促销海报，目标市场台湾
```

技能会自动：
1. 调研台湾 618 营销海报趋势
2. 输出调研报告并确认细节
3. 设计繁体中文文案
4. 生成海报图片
5. 验证并迭代优化

## 内置规范

### 台湾繁体本地化

| 简体 | 繁体 | 说明 |
|------|------|------|
| 优惠 | 優惠 | 标准繁体 |
| 活动 | 活動 | 标准繁体 |
| 体验 | 體驗 | 标准繁体 |
| 信息 | 資訊 | 台湾用语 |
| 秒杀 | 限時搶購 | 台湾用语 |
| 衔接 | 銜接 | 标准繁体 |

### Prompt 模板

技能内置标准化的 `GenerateImage` 提示词模板，包含：
- 品牌/卡通形象描述
- 文字内容和语言要求
- 色调和视觉风格
- 布局构图要求
- 禁止项（水印、多余文字等）

### 质量检查清单

| 检查项 | 标准 |
|--------|------|
| 品牌名称 | 正确显示，无多余图标 |
| 文字内容 | 无错别字，语言正确 |
| 卡通形象 | 清晰可辨 |
| 整体风格 | 符合调研建议 |
| 水印 | 无 AI 生成水印 |
| 布局 | 层次清晰，重点突出 |

## 适用场景

- 节日促销海报（618、双11、圣诞节等）
- 教育机构招生海报
- 品牌活动宣传海报
- 社交媒体营销图片（FB、IG、LINE）

## 注意事项

- 调研阶段搜索次数控制在 5 次以内
- 迭代上限 5 次，3 次无法解决同一问题时建议使用专业工具微调
- AI 生成图片可能出现文字渲染偏差，需要人工验证
