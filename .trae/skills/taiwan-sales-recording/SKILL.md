# 台湾销售团队录音下载与质检 Skill v2.0

## 核心定位
```
钉钉花名册 → CRM学员筛选 → 通话记录获取 → 录音批量下载
    → ASR转写 → AI质检 → 飞书文档报告 → 本地录音自动清理
```

## 核心特性

| 特性 | 说明 |
|------|------|
| **飞书文档报告** | 分析完成后自动生成飞书文档，包含完整质检报告 |
| **本地录音清理** | 分析完成后自动删除本地录音文件，节省存储空间 |
| **报告可追溯** | 删除录音后，报告仍包含CRM链接，可随时回听原始录音 |
| **关键信息完整** | 报告包含：学员ID、通话时间、员工姓名、质检评分等 |

## 决策树（快速入口）

```
收到录音相关请求
    │
    ├── 问"台湾销售录音""台湾CC录音"
    │       └─→ 进入完整流程（花名册→学员→通话→录音→报告）
    │
    ├── 问"某个员工的录音"
    │       └─→ 指定员工模式，跳过花名册获取
    │
    ├── 问"今日录音""昨天录音"
    │       └─→ 按日期筛选模式
    │
    └── 问"录音质检""话术分析"
            └─→ 下载后自动触发ASR+AI分析+飞书报告
```

## 触发后第一轮对话（必须确认）

1. **日期范围**：「要下载哪天的录音？」
2. **员工范围**：「要下载哪些员工的录音？」
3. **通话筛选**：「通话时长筛选条件？」
4. **下载数量上限**：「最多下载多少条？」
5. **是否分析**：「下载后是否进行ASR转写和AI质检？」

## 四阶段流程总览

| 阶段 | 内容 | 工具 | 输入 | 输出 |
|------|------|------|------|------|
| **第一阶段：花名册获取** | 从钉钉获取销售团队名单 | dws | 部门ID | 员工姓名列表 |
| **第二阶段：CRM查询** | 按员工查学员→查通话记录 | cc-workbench2 | 员工姓名 | 通话记录列表 |
| **第三阶段：录音下载** | 下载录音到临时目录 | cc-workbench2 | call_id | 录音文件(.wav) |
| **第四阶段：分析与报告** | ASR转写→AI质检→飞书文档→清理录音 | ASR+AI+lark-doc | 录音文件 | 飞书文档链接 |

## 存储策略

| 内容 | 存储位置 | 清理策略 |
|------|----------|----------|
| 本地录音文件 | `/tmp/taiwan_recordings/` | 分析完成后立即删除 |
| 临时文件 | `/tmp/` | 24小时后自动清理 |
| 飞书文档 | 飞书云文档 | 永久保留（可追溯） |
| 执行日志 | `~/.taiwan-recording/logs/` | 保留30天 |

## 配置文件

### config.json
```json
{
  "version": "2.0.0",
  "dept_ids": {
    "taiwan": ["需要从钉钉CLI查询获取"]
  },
  "defaults": {
    "date_range": "today",
    "min_call_duration": 30,
    "max_downloads": 50
  },
  "storage": {
    "temp_recording_dir": "/tmp/taiwan_recordings",
    "log_dir": "~/.taiwan-recording/logs"
  },
  "asr": {
    "enabled": true,
    "model": "paraformer-v2",
    "api_key": "YOUR_DASHSCOPE_API_KEY"
  },
  "feishu": {
    "enabled": true,
    "report_folder": "台湾销售录音报告"
  }
}
```

## 脚本文件

| 脚本 | 用途 |
|------|------|
| `src/get_roster.py` | 获取钉钉花名册 |
| `src/query_crm.py` | 查询CRM学员和通话 |
| `src/download_recordings.py` | 批量下载录音到临时目录 |
| `src/analyze_recordings.py` | ASR+AI质检分析 |
| `src/feishu_report.py` | 生成飞书文档报告 |
| `src/cleanup.py` | 清理本地录音文件 |
| `src/main.py` | 主控流程（整合全流程） |

## 安全与权限

- 钉钉CLI需要企业管理员授权
- CRM CLI需要登录状态（浏览器CDP）
- 飞书文档需要lark-cli授权
- 本地录音文件分析后自动删除，不占用存储
- 报告保存在飞书云端，永久可追溯
